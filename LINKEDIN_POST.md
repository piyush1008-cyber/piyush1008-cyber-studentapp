# LinkedIn Post

I’m excited to share my **Student Management System** project built with **Java, Spring Boot, Spring Security, JPA, JWT, and Thymeleaf**.

This project includes:

- secure JWT-based login
- protected student CRUD APIs
- course-wise filtering and topper lookup
- statistics endpoint for student insights
- browser dashboard for quick demo
- built-in docs page for easy project walkthrough

What I focused on while building it:

- backend security
- clean API design
- validation and error handling
- database integration
- project presentation and usability
